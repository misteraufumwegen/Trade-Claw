Semantic Color Usage

Trading Profit / Win
Positive P&L values, winning trades, trend indicators (↑), success messages
✓ WCAG AA compliant on dark background
Trading Loss / Error
Negative P&L values, losing trades, trend indicators (↓), error states
✓ WCAG AA compliant on dark background
Alert / Warning
Drawdown warnings, risk alerts, special emphasis, attention needed
✓ WCAG AA compliant on dark background
Action / Focus
Primary buttons, active tabs, focus rings, interactive elements, highlights
Use for high-emphasis UI
